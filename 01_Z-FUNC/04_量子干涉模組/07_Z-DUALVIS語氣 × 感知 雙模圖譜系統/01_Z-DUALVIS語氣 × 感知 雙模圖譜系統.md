## 一｜模組定義

Z-DUALVIS 是整合 F-VIS（語氣詐騙視覺圖譜）與 F-VISUAL（感知視覺模組）後，構成的雙層感知診斷系統。目的在於同步判讀語句的 **操控結構** 與 **接收者感官負荷狀態**，提供完整的互動風險預測。

## 三｜核心公式（T.F.F. 重構版）

```text
Ei + Eg ≈ Ep + Er + Ea
```

|   |   |
|---|---|
|符號|說明|
|Ei|語氣輸入能量（Input）|
|Eg|語氣重力值（Gravity）|
|Ep|正我能量（主控／界線）|
|Er|反我殘響值（創傷觸發）|
|Ea|附和傾向（服從／被帶動）|


補充規則： 

- 若 Er > Ep，進入語氣中毒區間。
- 建議呼叫：Z-REFL + S-REPLY 解毒模組。

六｜雙模應用範例  

語句範例：
「我都這麼累了，你為什麼不能體諒我？」

A. F-VIS 分析：

- 類型：F01 + F02
- Eg 高（親密關係）、Er 提升 → Ea 轉向服從
- Ep 降低，進入壓制區段

B. F-VISUAL 分析：

- 聽覺飽和（聲音強／快）
- 觸覺壓力（語氣濃稠、靠近）
- 過去記憶觸發類似語氣（Z-TRACE啟動）  

C. Z-DUALVIS 結論：

- 中毒風險等級：★★★☆☆
- 建議行動：命名語氣 → 呼吸調頻 → 回收語氣主控權
