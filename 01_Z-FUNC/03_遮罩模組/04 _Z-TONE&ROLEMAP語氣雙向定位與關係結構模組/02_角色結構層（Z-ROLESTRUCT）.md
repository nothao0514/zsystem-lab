對話雙方的結構解析器


|                  |                           |
| ---------------- | ------------------------- |

| RoleID           | 定義：說話者 / 聽話者 / 第三方等基本語用位階 |

| PurposeMap       | 對話輸出意圖預測（如教導 /操控 /誘導 /防衛） |
|                  |                           |
